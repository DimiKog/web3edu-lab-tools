

# 🧩 Identity Scope Visualizer
**Web3Edu — Lab Tool**

---

## 🇬🇧 English

### Overview
The **Identity Scope Visualizer** is an educational Web3 tool that demonstrates a core blockchain concept:

> **The same cryptographic identity (wallet address) can have different meaning, permissions, and trust depending on the network in which it exists.**

The tool helps learners understand that:
- a blockchain address is derived **only** from a private key
- the address itself does **not encode roles, permissions, or trust**
- those properties are defined by the **network context**

---

### Learning Objectives
After using this tool, learners should be able to:
- Explain why the *same address* can appear on multiple blockchains
- Distinguish between **cryptographic identity** and **identity meaning**
- Understand the concept of **identity scope**
- Compare identity behavior across:
  - permissioned networks (e.g. Besu)
  - permissionless networks (e.g. Ethereum)

---

### Tool Concept
The tool visualizes two different network contexts:

**Besu Edu-Net (Permissioned)**
- Role-based access control
- Network-defined permissions
- Institutional / academic trust model

**Public Ethereum (Permissionless)**
- Open participation
- No predefined identity roles
- Protocol-level governance

Both contexts show the **same wallet address**, interpreted differently by each network.

---

### How to Use
1. Open the tool (English or Greek version)
2. (Optional) Enter a wallet address in the input field
3. Click on each network card to expand details
4. Compare permissions, governance, and trust assumptions

> ℹ️ This tool is **conceptual** and does **not** connect to a live blockchain.

## 🇬🇷 Ελληνικά

### Περιγραφή
Το **Identity Scope Visualizer** είναι ένα εκπαιδευτικό εργαλείο Web3 που αναδεικνύει μια θεμελιώδη έννοια των blockchain:

> **Η ίδια κρυπτογραφική ταυτότητα (διεύθυνση πορτοφολιού) μπορεί να έχει διαφορετικό νόημα, δικαιώματα και επίπεδο εμπιστοσύνης ανάλογα με το δίκτυο στο οποίο χρησιμοποιείται.**

Το εργαλείο βοηθά τους εκπαιδευόμενους να κατανοήσουν ότι:
- μια διεύθυνση blockchain παράγεται αποκλειστικά από το ιδιωτικό κλειδί
- η διεύθυνση δεν ενσωματώνει ρόλους ή δικαιώματα
- το νόημα της ταυτότητας καθορίζεται από το **δίκτυο**

---

### Μαθησιακοί Στόχοι
Μετά τη χρήση του εργαλείου, οι εκπαιδευόμενοι μπορούν να:
- Εξηγήσουν γιατί η ίδια διεύθυνση εμφανίζεται σε διαφορετικά blockchain
- Διαχωρίσουν την **κρυπτογραφική ταυτότητα** από το **νόημά της**
- Κατανοήσουν την έννοια του **εύρους ταυτότητας**
- Συγκρίνουν συμπεριφορές ταυτότητας σε:
  - αδειοδοτημένα δίκτυα (π.χ. Besu)
  - δημόσια δίκτυα (π.χ. Ethereum)

---

### Εννοιολογική Προσέγγιση
Το εργαλείο παρουσιάζει δύο διαφορετικά δικτυακά πλαίσια:

**Besu Edu-Net (Αδειοδοτημένο)**
- Έλεγχος πρόσβασης βάσει ρόλων
- Κανόνες ορισμένοι από το δίκτυο
- Θεσμικό / ακαδημαϊκό μοντέλο εμπιστοσύνης

**Δημόσιο Ethereum (Μη αδειοδοτημένο)**
- Ανοιχτή συμμετοχή
- Χωρίς προκαθορισμένους ρόλους ταυτότητας
- Διακυβέρνηση σε επίπεδο πρωτοκόλλου

Και στα δύο πλαίσια εμφανίζεται η **ίδια διεύθυνση**, με διαφορετική ερμηνεία.
