# Web3Edu Lab Tools

This repository contains **interactive tools** used in the Web3Edu lab series.
These tools help students **visualize, verify, and explore** core Web3 concepts.

## Design principles
- One concept per tool
- Minimal UI
- Immediate feedback
- Network-aware (Besu-first)

## Tools
- Network Check — wallet & network verification
- Address Anatomy — visualizing Ethereum addresses
- Identity Scope — network-scoped identity

## Usage
Each tool can be used:
- standalone
- inside a lab
- in a classroom setting# web3edu-lab-tools
